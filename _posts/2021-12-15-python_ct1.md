## 풀이
~~~python
import heapq
def solution(sco, K):
    # heapq.heapify(sco)
    sco.sort()  # 시간이 더 적게 걸림
    count = 0
    
    while sco[0] < K:           
        if len(sco) <= 1:
            return -1
        else:
            heapq.heappush(sco, heapq.heappop(sco) + heapq.heappop(sco) * 2)  
            count += 1  
    return count
~~~ 

1. heapq 함수를 사용했습니다. 리스트를 우선 정렬하였는데 heapq.heapify()보다는 .sort()가 시간이 더 적게 걸리는 것을 확인하고 수정하였습니다.
