## 풀이
~~~python
def solution(n):
    count = 0
    
    for i in range(1, n+1):
        sum = 0     # 위치 중요함
        for j in range(i, n+1):
            sum += j
            if sum == n:
                count += 1
                break
            elif sum > n :
                break
    return count
~~~
