## 풀이
~~~python
def solution(participant, completion):
    answer = ''
    
    participant.sort()
    completion.sort()
    
    for i in range(1, len(completion)):
        if participant[i] != completion[i]:
            answer = participant[i]
            break
        elif participant[i] not in completion[i]:
            answer = participant[i]       
            break
    
        return participant[i]
~~~
