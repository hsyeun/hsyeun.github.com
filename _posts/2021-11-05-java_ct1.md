## 풀이
~~~java
import java.util.*;
public class Main {

	public static void main(String[] args) {
		Scanner sc = new Scanner(System.in);
		int n = sc.nextInt();
		int x = sc.nextInt();
		
		int[] a = new int[n];
		
		for(int i = 0; i<n; i++) {
			a[i] = sc.nextInt();
			
			if(a[i] < x) {
				System.out.print(a[i] + " ");
			}
		}
	}
}
~~~
